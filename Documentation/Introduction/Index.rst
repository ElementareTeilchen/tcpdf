Einführung
----------

Diese Erweiterung stellt lediglich einen Wrapper um TCPDF (PHP PDF Library) zur Verfügung und ermöglicht die einfache Installation innerhalb von TYPO3.

Die Erweiterung bietet dann eine einfache Möglichkeit TCPDF in eigenen Erweiterungen zu nutzen.

.. NOTE::
   * Diese Version stellt TCPDF in der Version 6.2.13 vom 10.06.2016 bereit.
   * Diese Version basiert auf dem Repository: https://github.com/tecnickcom/TCPDF .
   * Das Verzeichnis mit den Beispielen (examples) ist nicht Bestandteil dieser Erweiterung.

.. IMPORTANT::
   * TCPDF wird so nicht mehr weiter entwickelt, aber in verschiedenen Projekten noch eingesetzt.
   * Es wird an einem Nachfolger von TCPDF gearbeitet. Sollte die Neuimplementierung dann soweit entwickelt sein, dass damit gearbeitet werden kann, wird dieser neue Stand in dieser Erweiterung einfließen.
   * Sollte sich die Major-Version dieser Erweiterung ändern ist auf jeden Fall zu prüfen, welche Anpassungen vorgenommen werden müssen. Eine entsprechende Dokumentation versuche ich hier bereit zu stellen.
